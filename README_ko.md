## 프로젝트 소개

ActivityHub는 커뮤니티 플랫폼의 백엔드를 가정하여 설계한 데이터베이스 프로젝트입니다.

관계형 데이터베이스를 직접 설계하고, Microsoft SQL Server를 활용하여 데이터베이스를 구현하였으며, 실행 계획 분석과 인덱스 최적화를 통해 쿼리 성능을 개선하는 과정을 학습하는 것을 목표로 하였습니다.

### 주요 내용

- 관계형 데이터베이스 설계
- 데이터베이스 정규화
- Microsoft SQL Server
- 쿼리 성능 최적화
- 실행 계획 분석
- 인덱스 튜닝
- ASP.NET Core MVC 기반 관리자(CRUD) 페이지 구현

## 프로젝트 목표

- 정규화된 관계형 데이터베이스 설계
- Microsoft SQL Server 활용
- 실행 계획 분석
- 쿼리 성능 최적화
- ASP.NET Core MVC 기반 관리자 페이지 구현

## 테스트 데이터

쿼리 성능과 인덱스 효율을 검증하기 위해 테스트 데이터 자동 생성 스크립트를 구현하였습니다.

| 테이블 | 목표 데이터 수 |
|--------|---------------:|
| Posts | 100,000 |
| Comments | 500,000 |

스크립트는 부족한 데이터만 자동으로 추가하도록 구현하여 여러 번 실행해도 중복 데이터가 생성되지 않습니다.

또한 댓글은 기존 게시글에 분산하여 생성되도록 구성해 실제 서비스 환경과 유사한 형태로 쿼리 성능을 테스트할 수 있도록 하였습니다.
