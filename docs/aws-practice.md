# AWS Practice

## Goal
Learn the basic concepts of AWS cloud services and practice using EC2 and S3.

## EC2
- Created an Amazon Linux 2023 EC2 instance.
- Connected to the instance from macOS using SSH.
- Practiced basic Linux commands:
  - `whoami`
  - `pwd`
  - `ls`
  - `mkdir`
  - `cd`
  - `cat /etc/os-release`
- Learned the roles of EC2, EBS, and SSH.

## S3
- Created an S3 bucket.
- Uploaded and downloaded files.
- Learned the difference between EC2 (compute) and S3 (object storage).

## What I Learned
- Understood the basic workflow of launching and accessing a cloud server.
- Learned how to connect to a Linux server securely using SSH.
- Understood the roles of EC2, EBS, and S3 in AWS.
- Practiced basic cloud infrastructure concepts.
